# Socket Program in C

## How to use

### 1. Clone this repository

```shell
git clone https://github.com/1239uth/sockets.git
cd sockets
```

### 2. Build the code

```shell
./build.sh
```

### 3. Run the server

```shell
./run-server
```

### 4. Run the client (in another terminal)

```shell
./run-client
# defaults connecting to localhost:5432
```
